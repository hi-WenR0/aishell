## 🔧 技术实现

### 核心技术栈

- **语言**: TypeScript (ES2022)
- **CLI 框架**: Commander.js
- **UI 框架**: React 18 + Ink 5
- **API 集成**: Anthropic SDK, OpenAI SDK
- **环境管理**: dotenv

### 架构特点

1. **模块化设计**: 通过 Provider 接口实现模型解耦

4. **可扩展性**: 易于添加新的 AI 模型

### 1. 智能命令生成

